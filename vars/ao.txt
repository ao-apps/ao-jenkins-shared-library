# `setVariables(binding, currentBuild, scm, params)`
Sets all variables, providing defaults if not already set in the binding.

Variables that may be defined in the header of `Jenkinsfile`:

* `deployJdk`  
  The version of JDK that will be used for the deploy stage.  
  Defaults to '21'
