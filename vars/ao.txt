<a href="https://www.jenkins.io/doc/book/pipeline/shared-libraries/">Jenkins shared library</a> for all AO-supported <a href="https://www.jenkins.io/doc/book/pipeline/syntax/">Jenkins Pipelines</a>.

<h3><code>setVariables(binding, currentBuild, scm, params)</code></h3>
<p>
  Sets all variables, providing defaults if not already set in the binding.
</p>
<p>
  Variables that may be defined in the header of <code>Jenkinsfile</code>:
</p>
<dl>
  <dt><code>deployJdk</code></dt>
  <dd>
    The version of JDK that will be used for the deploy stage.<br>
    Defaults to '21'
  </dd>
</dl>
